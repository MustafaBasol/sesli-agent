## Dental CRM reuse requirement

This project must reuse suitable patterns from the user's Dental CRM project instead of rebuilding every integration and tenancy pattern from scratch.

Before implementing backend auth, tenant scoping, integration settings, WhatsApp, Instagram, message templates, provider abstraction, inbox, or automation jobs, inspect the Dental CRM codebase if available.
